# ✨Highlights
- Draggable, resizable, openable and closable windows.
- A working notes app that stores notes in local storage to be used again.
- A music player that utilizes the non copyrighted songs present in the repo.
- A real wikipedia page embed.
- A fake terminal that really does respond to some commands.


# 💁‍♂️Overview
**SimpOS** is a web page made to look like a desktop OS. It mainly uses resizable and draggable windows to mimic a real desktop environment. It comes with a top bar and really functional apps.

# 🚀Usage
To run your own instance locally, follow these steps:
1. Clone this repository with: <br>
```git clone https://github.com/Zyrox-exe/SimpOS.git```
2. Change the source code as per your liking.
3. Open `index.html` in any modern browser.
